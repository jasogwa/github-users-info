# Installation 

## Install with docker

* Install and configure docker on your machine
* At the root folder run `docker-compose up`

# Install without docker

* Recommended node version: 16.17.0

## BackEnd

*   ``cd BackEnd``
*   ``npm install``
*   ``npm run dev``

## FrontEnd

*   ``cd FrontEnd``
*   ``npm install``
*   ``npm run dev``

# Run

*   ``Open : http://localhost:8000/  on your browser``

# Testing

## FrontEnd testing

*   ``cd FrontEnd``
*   ``npm run test``

## BackEnd testing

*   ``cd BackEnd``
*   ``npm run dev``
*   ``npm run test``
